O que é o HTML?

O html (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web

Fonte: mdn web docs

"HiperTexto"

O HiperTexto refere-se aos links que conectam páginas da web entre si. Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na world wide web.

"Marcação"

O html usa "marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador da web. A marcação HTML inclui elementos especias.

CSS

Cascading Stylesheets — ou CSS — é a primeira tecnologia que você deve aprender após o HTML. Enquanto o HTML é utilizado para definir a estrutura e semântica do seu conteúdo, o CSS é usado para estilizá-lo e desenhá-lo.

Fonte: mdn web docs

O que é CSS?

O CSS é uma linguagem de folha de estilo composta por “camadas”, criado com o propósito de estilizar as páginas HTML, ou seja, definir a aparência das páginas, para deixá-las visualmente mais bonitas e agradáveis, podendo alterar a fonte, cor, posicionamento dos elementos, layout e muito mais.

Finalidade do CSS

Ou seja, CSS é uma linguagem para especificar como documentos são apresentados aos usuários — como eles são estilizados, dispostos etc.
